리눅스(혹은 유닉스나 맥) 유저들은 위 프로그램 설치가 비교적 더 용이하게 가능하다.

특히 Linux의 경우 한줄의 명령어로 많은 package들이 설치가능하다.

예를 들어 Ubuntu에서는 gfortran을

```bash
$> sudo apt-get install gfortran
```

위의 명령어로 설치할 수 있다.

LAPACK의 경우

```bash
$> sudo apt-get install liblapack-dev
```
로 간단히 설치 가능하다.
